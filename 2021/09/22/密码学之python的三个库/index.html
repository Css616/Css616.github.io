密码学之python的三个库

1.gmpy2

是python的一个扩展库

安装：

先在cmd中输入wheel检查一下是否安装了wheel文件包，若没有安装，则输入：pip install wheel

再安装gmpy2所需要的whl文件。点击进入https://www.lfd.uci.edu/~gohlke/pythonlibs/下载所需文件放到python文件目录下。

然后再cmd命令下输入pip install [whl文件的路径]安装whl文件包

输入pip install gmpy2

最后用import gmgy2命令测试一下，如果没有报错，说明安装成功。

注意：使用pip install时，用什么地方弹出的cmd命令行窗口都可以。但如果要对具体的文件进行操作，就该在具体文件所在目录下的顶部输入框中，敲入cmd再执行相关操作。
