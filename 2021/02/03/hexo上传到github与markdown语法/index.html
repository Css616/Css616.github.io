一、hexo上传到github

1.创建文件

右键打开”Git Bash Here”, 输入如下命令

hexo new “title”

在中

创建标题为title的文章

2.编写文章

3.部署发布

右键打开”Git Bash Here”,输入

hexo g部署

hexo d发布

二、markdown简单语法

markdown 编辑器：Typora

markdown语法：

1）标题

2）标记

3）分割线

4)图片

创建文章后，同时在中
