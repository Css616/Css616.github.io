渗透基础知识小补充

记一次国科渗透攻防项目实战课

​ in advance

补充知识：

VMWare （Virtual Machine ware)

是一个“虚拟PC”

可以在一台机器上同时运行更多Windows、DOS、LINUX系统

setup：安装

domain：域名

DNS：域名服务器（Domain Name System）

系统服务：电脑启动软件也启动

对应“非服务模式”：手动启动，不随电脑启动而启动

index：是名称，一般首页都命名为index

**url:**在www上，每一信息资源都有在网上唯一的地址，该地址就叫url，就是指网络地址。通俗讲，链接叫url

ip地址：IP地址由两部分组成，即网络地址和主机地址。

  1. 网络号 net-id，它标志主机（或路由器）所连接到的网络，网络地址表示其属于互联网的哪一个网络

  2. 主机号 host-id，它标志该主机（或路由器），主机地址表示其属于该网络中的哪一台主机

    延：192.168.1.1 掩码255.255.255.0，那么网络号就是192.168.1.0，也就是它的网段

C段：指的是同一内网段内的其他服务器，每个IP有ABCD四个段，举个例子，192.168.0.1，A段就是192，B段是168，C段是0，D段是1

而C段嗅探的意思就是拿下它同一C段中的一台服务器，也就是说是D段1-255中的一台服务器

Ping： 是 Windows 和 Linux 都自带的一个扫描工具，用于校验与远程计算机或本机的连接

主机存活：主机开启

网关：就是一个网络连接到另一个网络的“关口”，也就是网络关卡。

从一个房间走到另一个房间，必然要经过一扇门。同样，从一个网络向另一个网络发送信

息，也必须经过一道“关口“，这道关口就是网关。

备案号：是网站是否合法注册经营的标志，一个网站域名要上线必然要经过备案程序

linux系统:

一.linux的主要目录

1./etc–系统配置文件存放的目录

2./bin–保存可执行二进制文件的目录，如ls、tar、mv、cat

3./–根目录（在linux下有且只有一个根目录。然后在/目录在有子目录如/root、/home、/etc/、/bin等

4./home–系统默认的用户家目录

二.linux快捷键

放大终端 Ctrl Shift +

打开终端Ctrl+Alt+t或者右键点击 出现的便是shell界面

类似windows操作系统下的终端命令行

即cmd界面

按下“确定”按钮 便弹出一个黑色的终端软件

如果要看当前的用户主机名则输入whoami命令，则显示：主机名\用户名

如果我们要看看自己电脑的网络配置啊ip地址啊，可以输入ipconfig命令，并按下回车

三、linux系统安全工具之Nmap

Nmap（Network Mapper）是一款开放源代码的网络探测和安全审核的工具。它使用原始IP来发现网络上有哪些主机，那些主机提供什么服务(应用程序名和版本)，那些服务运行在什么操作系统(包括版本信息)，它们使用什么类型的报文过滤器/防火墙。

总的来说其基本功能有四个：

  1. 发现主机

  2. 扫描端口

  3. 侦测版本

  4. 侦测操作系统

Nmap所识别的几个端口状态:

open –开放 closed –关闭

filtered –被过滤

open|filtered –开放或者被过滤

信息收集与Nmap

渗透测试思路

1.信息收集（前提）

2.漏洞挖掘：AWVS payload验证

3.漏洞利用：写payload或者利用工具（sqmap、xsser、c刀）

4.输出渗透测试报告

渗透测试有两种方法

1.白盒（代码审计）（通过代码判断是否存在隐患）（在内网情况下）

2.黑盒（模拟黑客去渗透测试）（在外网情况下如何攻击到服务器里面）

渗透测试之信息收集

1.基本信息

1）域名信息（示例：goktech.cn）–站长之家–http://whois.chinaz.com/

2）子域名(示例：bbs.goktech.cn) –https://phpinfo.me/domain/ (在线子域名查询)

​ –Layer子域名挖掘机4.2 纪念版

3）ip地址 –站长之家–http://ip.tool.chinaz.com/

4）同ip的其他网站（旁站）信息 –站长之家–http://stool.chinaz.com/same

​ –https://www.webscan.cc/

5）同网段的其他服务器（c段）信息 c段常用在护网或攻防中

​ –https://www.webscan.cc/

例：宿舍几台计算机所处的环境就是同网段，是一个小型的g网

6）端口信息：80（80端口对应http协议）

​ 443（https协议）

​ 445（445端口对应共享文件夹） –Nmap

7）服务（web服务即网页服务），服务版本（version） –Nmap

8）信息泄露（密码收集） –https://github.com/

9）网站目录 –御剑

10）业务信息 –搜索引擎，如：谷歌、百度

11）备案号 –http://icp.chinaz.com/

12）快速切换搜索引擎获得搜索结果 –听潮社区URL采集工具2.3

注：

旁站的利用：旁站指的是同一服务器上的其他网站，很多时候，有些网站可能不容易入侵。那么，可以查看该网站所在的服务器上是否还有其他网站。如果有其他网站的话，可以先拿下其他网站，然后再拿到服务器的权限，最后就自然可以拿下该网站。

故不管攻击哪一个网站，都是同一台服务器。因为同ip意味搭建在同一台服务器上。

报错信息的利用：也许会涉及敏感信息。因为不同的中间件有不同的漏洞

端口的利用：通过服务器端口是否开放判断有无提供对应的服务。如开放80端口便是提供一个网页的服务。故也可能存在漏洞。

但是端口可以被修改。如可以将http对应的端口改为8080端口。即默认是80，可以人为修改。故80端口不一定是http。所以要收集服务。

备案号的利用：利用备案号反向去查目标的所有合法网站。

2.os版本

​ –Nmap工具（kali要有ip地址，并且目标服务）

操作系统（operation system，简称OS），包含windows server、Linux、mac

3.web中间件（web程序，用来提供网页服务）

类型、版本：iis、apache、nginx、weblogic

4.数据库

类型、版本：mysql、mssql、oracle、access

5.后端的脚本类型

php、jsp、asp、aspx

C语言的注意点们

诞生自教训中的C语言的注意点们

1.一字节=8个二进制位 无符号数一定为整数

2.用/运算时：整数/整数=整数（舍去小数的整型数）

用%运算时：只能用于整数 符号与%左侧符号相同

用int运算时：int(number)求不大于number 的最大整数，int(3.8)=3，int(-3.8)=-4

3.所有变量应赋初值，给变量赋初值需取地址&

4.scanf()是格式输入函数 scanf(格式控制序列，地址序列);

注1:如(“a=%d…”,&a,&…);

注2:d为十进制整数 格式说明符由%接格式字符组成

5.强制类型转换 格式:(类型名)(表达式)

注1：单个变量可以不加括号

注2：不改变变量类型 如(double)a只是将变量a的值转换成一个double型的中间量 a的数据类型未转换成double型

6.switch语句 实现多路分支选择避免嵌套的if结构分支过多

格式 switch(整型表达式){

​ case 整型常量表达式1: 语句组1;break;

​ ……

​ case 整型常量表达式n: 语句组n;break;

​ [default: 语句组n+1]

​ }

注1：整型常量表达式可是字符型常量（字符型常量以ASCⅠⅠ码值表示），为整型值。同理，整形表达式可是字符型变量组成的表达式只要有一个整型值

7.自增运算符++ 单目运算符 优先级高于所有双目运算符 自右向左结合

++i 使用i前 先i加1

i++ 使用i后，i加1
